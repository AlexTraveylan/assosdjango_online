Pour creer votre espace dans ce site qui permet de gerer votre association de parents d'élèves, il vous suffit de connaitre le RNA de votre association, on s'occupe du reste.
C'est le numero de votre association qui se trouve dans les documents de sa création, sinon vous pouvez le retrouver sur le site officiel du gouvernement à l'adresse suivante : journal-officiel.
Une fois votre numéro et votre association crée, il vous suffira de rejoindre la page crée pour vous.
Puis de de demander aux membres de votre association de la rejoindre aussi.
Vous pourez tous ensemble pour désigner le membre président, une fois fait, vous pourez profiter de toutes les fonctionnalité du site.
Il pourra aussi distribuer aux autres membres les droits (ou pas) pour gerer le site avec lui.
